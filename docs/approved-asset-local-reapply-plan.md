# Approved Asset Local Reapply Plan

## Branch
- `feature/visualreapply`

## Objective
Add a local reapply layer so the God Mode can take a mobile visual approval or revision decision and turn it into a concrete local reapply step on the PC for the selected project asset.

## Scope
- represent approved asset reapply sessions
- represent visual revision apply actions
- expose a compact package for the mobile cockpit
- expose the next local reapply action
- prepare the next phase of tighter asset iteration inside the local build pipeline
