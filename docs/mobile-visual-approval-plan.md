# Mobile Visual Approval Plan

## Branch
- `feature/visual-approval`

## Objective
Add a visual approval layer so the mobile cockpit can display previews from the PC, receive quick approval or revision decisions, and keep a short visual iteration loop inside the local first runtime.

## Scope
- represent visual approval sessions by project and asset
- represent approval and revision decisions
- expose a compact package for the mobile cockpit
- expose the next visual approval action
