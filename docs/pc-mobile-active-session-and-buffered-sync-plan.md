# PC Mobile Active Session And Buffered Sync Plan

## Branch
- `feature/pcmobilesession`

## Objective
Add an active PC plus mobile session layer so the God Mode can represent link state, queued commands, buffered results and flush readiness between the local PC brain and the mobile cockpit.

## Scope
- represent active pc mobile sessions
- represent buffered command and result sync state
- expose a compact package for the mobile cockpit
- expose the next session or buffer action
- prepare the next phase of stronger live control and recovery handling
