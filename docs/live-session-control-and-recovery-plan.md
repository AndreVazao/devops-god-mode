# Live Session Control And Recovery Plan

## Branch
- `feature/livesession`

## Objective
Add a live session control and recovery layer so the God Mode can represent active control decisions, temporary degradation handling, and automatic recovery steps between the local PC brain and the mobile cockpit.

## Scope
- represent live session control states
- represent recovery actions after link degradation or interruption
- expose a compact package for the mobile cockpit
- expose the next control or recovery action
- prepare the next phase of stronger end to end remote operation resilience
