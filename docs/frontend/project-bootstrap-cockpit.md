# Project bootstrap cockpit

## Routes

- `/api/project-bootstrap-cockpit/dashboard`
- `/app/project-bootstrap`

## Objective

Give one go-no-go cockpit for launching a new project by combining diagnostics, remediation and provider onboarding into one readiness view.

## What it shows

- launch readiness flag
- readiness score
- blockers merged from diagnostics and onboarding
- launch phases from diagnosis to project launch
- remediation actions still pending
- resulting repo plan for the target project
