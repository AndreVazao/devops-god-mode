# Frontend Chat Cockpit Plan

- `feature/sessionresilience`

Start reshaping the frontend into a chat-first cockpit that feels closer to a real assistant workspace, with continuous conversation flow, new chats, history, rename actions and approval popups.

- chat style cockpit shell
- sidebar with new chats and history
- rename conversation modal
- approval and quick decision popup layer
- compact layout ready for PC first and mobile follow-up adjustments
