# GOD MODE - MEMÓRIA MESTRE

## Objetivo
Aplicação Android com botões flutuantes sobre outras apps, ligada a várias IAs, GitHub e memória Obsidian, para ajudar André Vazão a criar, alterar, gerir e continuar projetos diretamente pelo telemóvel.

## Plataforma inicial
Android.

## Utilizador principal
André Vazão.

## Conceito central
O God Mode funciona como painel flutuante universal. Ele acompanha o utilizador enquanto trabalha no telemóvel, captura contexto, consulta memória externa, conversa com IAs e executa ações aprovadas.

## Funcionalidades principais
- Botões flutuantes sobre outras apps
- Captura de contexto do ecrã
- Ligação a ChatGPT, Gemini, Claude ou outras IAs
- Ligação ao GitHub
- Criação de repositórios
- Alteração de ficheiros
- Geração de commits
- Consulta da memória Obsidian
- Escrita automática de logs
- Sistema de aprovação antes de ações perigosas
- Organização por projeto
- Histórico persistente
- Decisões persistentes
- Continuidade entre sessões

## Regras obrigatórias
- Nunca executar ação destrutiva sem aprovação explícita.
- Nunca guardar credenciais ou segredos técnicos no Obsidian.
- Guardar sempre decisões importantes em DECISOES.md.
- Guardar sempre alterações feitas em HISTORICO.md.
- Guardar sempre o estado final de cada sessão em ULTIMA_SESSAO.md.
- Antes de responder sobre um projeto, consultar primeiro a memória desse projeto.
- Antes de alterar código, consultar ARQUITETURA.md, DECISOES.md e BACKLOG.md.
- O utilizador trabalha muitas vezes só pelo telemóvel.
- O sistema deve ser prático, automático e visual.
