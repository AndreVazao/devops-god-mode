# BOT_FACTORY - MEMÓRIA MESTRE

## Objetivo
Criar uma plataforma que ajuda o God Mode a construir outros bots através de observação, análise contínua de jogos, engenharia reversa assistida, extração de padrões e geração de projetos de bot por fases.

## Utilizador principal
André Vazão.

## Conceito central
O telemóvel pode ser usado para jogar normalmente. O sistema observa o comportamento, organiza eventos, ações, ecrãs, padrões e objetivos do jogo. Depois o God Mode usa essa memória para planear e construir bots específicos para cada jogo.

## Importante
Este projeto ainda deve ser tratado como produto a mapear/auditar. Não é para desenvolver o bot completo nesta fase. O God Mode deve primeiro verificar se já existe repo/app. Se não existir, deve propor criação de repo novo com aprovação explícita.

## Funcionalidades desejadas
- Observação assistida de jogos
- Registo de ações do jogador
- Captura de padrões de ecrã e fluxo
- Organização por jogo
- Geração de backlog técnico por jogo
- Planeamento de bots específicos
- Reaproveitamento de arquitetura entre bots
- Aprendizagem contínua com aprovação
- Segurança e respeito por aprovações antes de automações sensíveis

## Regras obrigatórias
- Não guardar credenciais ou segredos técnicos.
- Não executar ações destrutivas sem aprovação explícita.
- Separar observação, análise, planeamento e execução.
- Guardar decisões em DECISOES.md.
- Guardar histórico em HISTORICO.md.
- Atualizar ULTIMA_SESSAO.md no fim de cada fase.
