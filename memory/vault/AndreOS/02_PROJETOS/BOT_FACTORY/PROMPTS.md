# BOT_FACTORY - PROMPTS

## Prompt base

Antes de desenvolver Bot Factory, verificar se já existe repo/app. Se não existir, propor criação de repo novo.

## Regra de continuação

Separar observação, análise, planeamento e execução.

## Instrução permanente

O God Mode deve tratar o Bot Factory como plataforma para criar outros bots por fases, começando por mapear o projeto, verificar se já existe repo/app e só depois propor desenvolvimento com aprovação explícita.
