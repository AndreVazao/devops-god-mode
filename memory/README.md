# AndreOS Memory Core

Memória externa persistente do God Mode, compatível com Obsidian.

## Objetivo

Guardar decisões, histórico, backlog, arquitetura, prompts, erros e última sessão por projeto, em Markdown, sem depender apenas do contexto curto das conversas.

## Regra principal

Nunca guardar passwords, tokens, chaves API, cookies, credenciais ou segredos nesta pasta. Secrets ficam sempre fora do Obsidian.

## Vault

O vault principal é `memory/vault/AndreOS`.

## Projetos base

- GOD_MODE
- PROVENTIL
- VERBAFORGE
- BOT_LORDS_MOBILE
- ECU_REPRO
- BUILD_CONTROL_CENTER
